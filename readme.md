## Installation 

Use the package manager [npm](https://docs.npmjs.com/try-the-latest-stable-version-of-npm) to install node_modules required for this app.
Run in ```root``` folder following commands

```bash
npm install 
```
then 
```bash
npm start
```
to start webpack dev server. Application will be running on ```http://localhost```


## Testing
To run client app tests go to ```root``` folder and run following command
```bash
npm run test
```
